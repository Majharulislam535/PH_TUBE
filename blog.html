var: Function-scoped, not block-scoped.Variables declared with var are hoisted to the top of their scope. This means that you can use the variable before it's declared in the code.
let: Block-scoped. Variables declared with let are only accessible within the block (or statement) where they are defined. Variables declared with let are hoisted but not initialized. Attempting to access the variable before the declaration results in a ReferenceError.
const: Block-scoped. Like let, const is hoisted but not initialized. Attempting to access the variable before the declaration results in a ReferenceError.

Null null is often used to explicitly indicate that a variable has no value or that an object property does not exist. It can be assigned as a default value when you want to indicate the absence of an object or value.
Undefined Variables that are declared but not assigned a value are automatically assigned the value undefined.If a function is called with fewer arguments than declared parameters, the missing parameters are set to undefined.

REST API A REST API is like a set of rules for how computer programs can talk to each other on the internet. It follows simple principles, such as using standard web addresses (URLs) to represent things and using standard actions like "GET" (get information), "POST" (send information), "PUT" (update information), and "DELETE" (delete information). This helps different programs work together smoothly. REST APIs are widely used in web development to make apps and websites communicate effectively.
